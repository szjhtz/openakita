# 秒哒 (Miaoda)

支持通过自然语言对话完成应用的创建、查看、修改、发布上线等操作。

## 支持应用类型

- 网页应用
- 微信小程序
- 游戏
- AI 工具
- SaaS 产品
- 数据看板

## 安装

通过 ClawHub 安装：clawhub install miaoda-app-builder

## 使用

直接用自然语言描述想要创建的应用，秒哒会自动生成代码并部署。
