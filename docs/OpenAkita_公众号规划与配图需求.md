# OpenAkita 公众号内容规划 & 配图需求

---

## 一、公众号定位与栏目规划

### 账号定位

**名称**：OpenAkita 开源 AI 助手
**定位**：开源多 Agent AI 助手的官方阵地，发布产品动态、功能教程、使用场景、技术分享
**调性**：专业但不枯燥，偏技术科普 + 实用教程风格

### 固定栏目（4 个）

| 栏目 | 发布频率 | 内容 | 适合受众 |
|------|----------|------|----------|
| **🔥 功能揭秘** | 每周 1 篇 | 单个核心功能的深度图文介绍 | 所有人 |
| **📖 上手教程** | 每周 1 篇 | 从安装到某个场景的完整教程 | 新用户/效率党 |
| **🛠️ 技术幕后** | 每两周 1 篇 | 架构设计、技术选型、开发故事 | 开发者 |
| **📢 版本动态** | 随版本发布 | 更新日志 + 亮点功能 + 下载链接 | 老用户 |

## 五、设计规范（给设计参考）

### 品牌色

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | 与 OpenAkita logo 一致 | 科技蓝系，品牌识别 |
| 辅助色 | 渐变紫蓝 | 背景、装饰 |
| 强调色 | 亮绿 / 橙 | 按钮、标注、✅ 图标 |
| 深色背景 | #0D1117 或 #1A1B2E | 科技感场景图 |
| 浅色背景 | #F8F9FA | 教程截图衬底 |

### 字体建议

- 标题：思源黑体 Bold / Noto Sans SC Bold
- 正文：思源黑体 Regular
- 英文/数字：Inter / SF Pro

### 风格关键词

**科技感但不冰冷 · 专业但不枯燥 · 扁平插画优先 · 圆角卡片 · 渐变背景**

### 截图美化规范

所有产品截图统一处理：
1. 加 **12px 圆角**
2. 加 **浅色投影**（0 4 12 rgba(0,0,0,0.1)）
3. 深色模式截图 → 深色衬底；浅色模式截图 → 浅色衬底
4. 标注用 **圆角矩形框 + 引出线 + 文字说明**，框线颜色用强调色
